See it running here: https://screen.studio/share/nKKDoe52

I led a demo-driven launch for an AI-powered UI generation tool, focusing on showing real-world value instead of just describing features.

Rather than starting with documentation, I built a working application to demonstrate how AI can generate production-quality interfaces when paired with real component libraries and contextual input. I collaborated closely with product and engineering to shape the experience, refine prompts, and highlight where the system meaningfully contributes—like layout, component selection, and theming.

The project evolved into a set of demos, videos, and live presentations used across conferences and developer outreach. It helped shift the conversation from “AI novelty” to practical workflows, while also generating valuable user feedback that informed improvements to the product and onboarding experience.
